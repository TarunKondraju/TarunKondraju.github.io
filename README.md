## ⚙️ How This Website is Built

This documentation-style portfolio is statically generated and automatically deployed. It is built using a modern, lightweight tech stack:

- **[MkDocs](https://www.mkdocs.org/):** A fast, simple static site generator that builds HTML from Markdown files.
- **[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/):** The premium, highly-customizable theme powering the responsive UI.
- **[GitHub Pages](https://pages.github.com/):** Free and reliable web hosting directly from this repository.
- **[GitHub Actions](https://github.com/features/actions):** The CI/CD pipeline that automatically compiles and deploys the website to the live server within minutes of any code commit.
