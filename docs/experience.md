## Products Developed

* **PUSAECMS:** A Google Earth Engine-based application for efficient big data processing for proving near real-time crop health management using vegetation indices, water indices, engineered composite indices, physical process biophysical variables estimation. The application allows the user to perform quantitative analysis and visualize temporal changes occurring in crop health at any given location.
* **VASUDHA:** An application to monitor soil health using by N, P, K, OC, EC, and pH estimated from VNIR/SWIR and MIR spectra generated from spectroradiometer and FTIR respectively in real-time. The application can also be used in offline mode to assess soil quality from large quantity of spectral data in an instance.
* **PUSAMIA:** An application to co-register, layer stack and assess proximal multispectral images of crops.
* **PAUDHA:** An application to assess crop health from large amounts of data provided by spectroradiometer in real-time or offline mode.
* **EdgeProcessing:** A pipeline to assess crop urea requirement from UAV multispectral data provided in real-time.
* **FieldMapper:** An Applcation for Constructing Digital Surface Models, Digital Terrain Modeks, Pointclouds, Multiband orthomosaics of RGB, Multispectral and thermal images captured from an UAV.
