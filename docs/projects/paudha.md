# PAUDHA: Plant Analysis Using Digital Hyperspectral Analytics

![Project overview image](../assets/images/paudha-thumb.png)

## Overview

PAUDHA is an analytical tool built to evaluate crop health from massive datasets generated by field spectroradiometers. It scales up plant chlorophyll retrieval and stress detection through rapid spectral processing.

**Study Area:** Global Application  
**Duration:** 2022 – Present  
**Role:** Lead Developer  
**Status:** Operational

---

## Methods & Tools

**Data Sources**

- Proximal hyperspectral point data from field spectroradiometers.
- Associated biochemical lab data for model training.

**Processing Steps**

1. Rapid ingestion of high-volume spectroradiometer text/CSV outputs.
2. Signal noise reduction and spectral derivative analysis.
3. Application of physical radiative transfer models and machine learning (e.g., Gaussian Process Regression).
4. Output of crop health classifications and trait retrievals.

**Tools Used**

| Tool | Purpose |
|------|---------|
| Field Spectroradiometers | Capturing continuous hyperspectral signatures |
| Python | Statistical modeling and Gaussian Process Regression |
| Data Visualization Libs | Generating spectral curve plots and health reports |

---
## Key Findings

- Optimized the retrieval of wheat crop traits (like Leaf Nitrogen Content, and Leaf Chlorophyll Content) from hyperspectral spectra.
- Provided a robust solution capable of operating in real-time in the field, or offline for post-campaign bulk analysis.
- Bridged the gap between raw proximal sensor data and actionable agronomical metrics.
