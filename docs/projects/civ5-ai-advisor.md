# Civ 5 AI Grandmaster Advisor 🧠🏛️

A powerful, standalone companion application for **Sid Meier's Civilization V** that connects your live game state to local or remote Large Language Models (LLMs).

## Features
- **Real-Time Map Extraction**: Renders the discovered map in a beautiful, interactive web UI outside the game.
- **Comprehensive Game State Logging**: Exports an extremely detailed snapshot of the world including exact enemy troop locations, hitpoints, and city yields.
- **AI Integration**: Connects directly to **Ollama** or **OpenAI** to feed the live game state to an LLM.
