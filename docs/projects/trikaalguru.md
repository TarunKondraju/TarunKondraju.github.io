# TrikaalGuru (Cosmic Blueprint) 🌌🕉️

![TrikaalGuru UI](../assets/images/trikaalguru-thumb.png)

A comprehensive Vedic Astrology dashboard that calculates precise planetary positions, zodiac divisional charts (Vargas D1 to D60), and Vimshottari Dashas, alongside an interactive AI Vedic Astrologer chatbot and professional PDF report export.

## Features
- **Vedic Chart Computations**: Calculates planetary positions and 16 divisional charts (Rasi, Navamsa, Dashamsha, etc.) using a Python-based Swiss Ephemeris engine wrapper.
- **Kundali Matching (Guna Milan)**: Analyzes compatibility between couples, computing scores out of 36 points and providing detailed breakdowns of all 8 Ganas (Varna, Vashya, Tara, Yoni, Graha Maitri, Gana, Bhakoot, Nadi).
- **AI Vedic Astrologer Chat**: Unlocks an interactive chatbot powered by Gemini 2.5 Flash API that acts as an expert Jyotishi, answering questions based on your unique charts.
- **Professional PDF Export**: Generates and prints multi-page PDF reports with Ganesha cover art, charts grid, and planetary tables.
- **Secure Payment Integration**: Fully integrated with Razorpay API (₹100 paywall) to safely unlock premium chat questions.

---

## Technical Stack
- **Languages:** Python, JavaScript
- **Frameworks:** FastAPI, Uvicorn, SQLite
- **APIs:** Google Gemini (genai SDK), Razorpay
- **Geospatial & Astrological Engine:** PyJHora (Swiss Ephemeris Wrapper)
